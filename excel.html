EXCEL

O Excel é um programa informático desenvolvido e distribuído pela Microsoft Corp. Trata-se de um software que permite realizar tarefas contabilísticas e financeiras graças às suas aplicações para criar e trabalhar com folhas de cálculo.

10 Atalhos do Excel
Ctrl W Feche uma pasta de trabalho
Ctrl A Abra uma pasta de trabalho
Alt+H Acesse a guia Página Inicial
Ctrl+B Salve uma pasta de trabalho
Ctrl+C Copiar seleção
Ctrl+Z Desfazer a ação recente
Alt+C, R Escolha uma cor de preenchimento
Ctrl+X Corte a seleção
Ctrl N Aplicar negrito
Alt+H, A, C Centralizar o conteúdo da célula
